# Pet Sanctuary Bot

## 🚀 Overview

A Discord bot built using **JavaScript**, **Node.js**, and **Discord.js** that aims to create a text-based sanctuary and progression game inside Discord servers.

Players will be able to earn coins, gain experience, level up, complete goals, unlock rewards, and grow their sanctuary over time. The project is designed with long-term progression and engaging gameplay in mind.

---

## 🔍 Features

* 🤖 **Discord Bot Integration:** Fully integrated with Discord using Discord.js.
* 💰 **Economy System:** Earn and spend coins through various activities.
* ⭐ **XP & Leveling:** Gain experience and unlock new progression opportunities.
* 🎯 **Goal System:** Complete objectives and milestones for rewards.
* 🛒 **Shop System:** Purchase items and upgrades using in-game currency.
* 🏆 **Achievements:** Unlock special rewards through gameplay progression.
* 📈 **Long-Term Progression:** Continuously improve and expand your sanctuary.

---

## 🛠️ Technologies Used

* **JavaScript** for bot logic
* **Node.js** as the runtime environment
* **Discord.js** for Discord API interaction
* **dotenv** for environment variable management

---

## ⚙️ How It Works

1. The bot connects to Discord using a secure bot token.
2. Players interact with the bot through Discord commands.
3. Commands award coins, experience, and progression rewards.
4. Player data is tracked to support leveling and goal completion.
5. New features, rewards, and progression paths unlock as players advance.

## 🏃‍♀️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/LeosThrone/pet-sanctuary-bot.git
   ```

2. Navigate to the project folder:

   ```bash
   cd pet-sanctuary-bot
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Create a `.env` file:

   ```env
   TOKEN=YOUR_DISCORD_BOT_TOKEN
   ```

5. Start the bot:

   ```bash
   node index.js
   ```
